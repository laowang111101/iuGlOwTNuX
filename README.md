# 超市管理系统 java1103

## 项目概述

本项目为超市管理系统，基于B/S架构和MVC模式，采用前后端分离的方式进行开发。系统结合了Java和Vue的技术优势，提供了一套完整的解决方案。

## 技术栈

- **后端：**
  - Java
  - SpringBoot
  - Mybatis-Plus
  - Redis
  - Mysql

- **前端：**
  - Vue

- **开发环境：**
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql

- **系统环境：**
  - Windows

## 功能模块

- **销售管理**
  - 销售主页
  - 销售记录
  - 积分兑换记录

- **人事管理**
  - 部门管理
  - 员工管理

- **库存管理**
  - 仓库管理
  - 出库明细
  - 入库明细
  - 供货商信息
  - 库存统计

- **会员管理**
  - 会员信息管理

- **系统管理**
  - 角色管理
  - 菜单管理

- **商品管理**
  - 分类管理
  - 商品信息
  - 积分商品
  - 销售统计

## 系统角色

管理员可以自由设置角色并分配权限。
